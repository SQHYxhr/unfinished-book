# 文案与内容规范

## 文案气质

- 诗意，但不晦涩。
- 安静，但不冷漠。
- 克制，但不故作高深。
- 文学，但必须清楚。

用户应该能理解每一个句子。如果一段话需要反复揣摩才能明白基本意思，它就不适合放在界面上。

## 推荐词汇

以下词汇与项目气质一致，鼓励使用：

暗处 · 显影 · 余温 · 影子 · 水面 · 光 · 花 · 门 · 旧纸 · 潮汐 · 尚未写完 · 合上 · 留下 · 看见

## 谨慎使用词汇

以下词汇容易引发误解或过度联想，如非必要不使用：

命运 · 灵魂 · 亡者 · 遗骸 · 神谕 · 禁忌 · 召唤 · 宇宙指引 · 神秘力量

如果必须使用，需在上下文中给出具体、有限度的解释，避免神秘化。

## 功能命名规范

界面上的操作按钮和链接应使用以下命名：

| 功能 | 按钮文案 |
|------|----------|
| 进入提问 | 写下一个问题 |
| 开始抽牌 | 让图像显影 |
| 保存记录 | 收进旧问录 |
| 关闭/退出 | 就此合上 |
| 删除记录 | 擦去这一页 |

不使用「开始占卜」「查看命运」「揭示答案」「塔罗预测」等措辞。

## 占卜内容边界

占卜内容（卡牌解释、结果展示）必须遵守以下边界：

- 仅用于娱乐、创作与自我反思。
- 不提供医疗、法律、财务建议。
- 不预测死亡时间。
- 不预测疾病结果。
- 不制造焦虑依赖。
- 不使用绝对化结果（如「你将会……」「注定……」）。
- 不鼓励用户将抽牌作为重大决策的唯一依据。
- 牌面解释应保留开放性，引导用户自己思考，而非替用户做决定。

## 死亡文化内容边界

「告别志」中的死亡文化内容必须遵守以下边界：

- **不猎奇**：不以震惊或吸引眼球为目的选择主题。
- **不消费悲伤**：不以情绪剥削的方式呈现他人的丧失。
- **标题克制**：不使用「最恐怖」「最诡异」「最离奇」等标题。
- **必须标注来源**：每篇文章应列出可查证的参考资料。
- **必须说明地区、时代、宗教、群体差异**：不将任何文化实践描述为统一的、不变的。
- **不自动编造资料**：不虚构学者、机构、书籍、统计数字、田野调查。
- **不浪漫化自杀和自残**：涉及相关主题时保持中立、谨慎，不美化。

## 文章组织原则

「告别志」文章以问题和物件为组织线索：

- **问题导向**：每篇文章围绕一个具体问题展开（如「某人去世后，ta 的衣物去了哪里？」）。
- **物件导向**：以具体物件（信、镜子、戒指、船等）作为叙事的锚点。
- **不以国别清单组织**：不以「X 国的死亡习俗」为标题结构。

## 语气检查清单

写完界面文案后，检查：

- [ ] 用户能一遍读懂吗？
- [ ] 有没有不必要的恐惧或焦虑暗示？
- [ ] 有没有绝对化的判断？
- [ ] 有没有伪造或模糊的来源暗示？
- [ ] 语言是邀请性的，还是命令性的？
- [ ] 留下的是余温，还是寒意？

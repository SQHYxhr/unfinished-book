# 视觉设计系统

## 视觉总方向

黑暗油画 × 超现实静物 × 当代编辑设计

## 视觉比例

| 类型 | 占比 | 说明 |
|------|------|------|
| 黑暗静物油画 | 60% | 主视觉背景、牌面图像、氛围载体 |
| 超现实遗物 | 25% | 物件象征变化、意象连接 |
| 腐朽浪漫主义场景 | 15% | 扩展场景、季节主题 |

## 色彩系统

以下 CSS 变量定义项目色彩。本阶段仅记录，不写入 globals.css。

```css
--void-brown: #0D0B09;       /* 最深背景 */
--gallery-black: #13110F;    /* 主背景 */
--varnish-brown: #241C16;    /* 次级背景、卡片 */
--bone-paper: #E3DAC9;       /* 正文白 */
--old-canvas: #B9A98F;       /* 辅助文字、分隔线 */
--dried-wine: #672F34;       /* 少量强调 */
--oxidized-gold: #9A7B47;    /* 标题点缀、图标 */
--deep-moss: #37463B;        /* 链接、辅助色 */
--night-blue: #1D2631;       /* 深夜氛围、阴影 */
```

## 使用原则

- 主背景使用黑褐（`--void-brown` / `--gallery-black`），不使用纯黑 `#000`。
- 正文使用骨白（`--bone-paper`）或旧画布色（`--old-canvas`）。
- 暗红（`--dried-wine`）只用于少量强调，不出现在大面积区域。
- 金色（`--oxidized-gold`）应像老画框或氧化金属，不闪亮不饱和。
- 不使用大面积紫色渐变。
- 正文区域不直接覆盖在复杂油画上——文字有独立的阅读层。
- 油画用于情绪，界面用于秩序。

## 字体原则

### 中文

- **标题**：文学感较强但可读的宋体或衬线字体（如 Noto Serif SC、Source Han Serif）。
- **正文**：清晰现代的中文黑体（如 Noto Sans SC、Source Han Sans）。

### 英文

- **辅助标题**：大写、较宽字距（letter-spacing: 0.15em–0.25em）。
- **正文**：与中文正文字体协调的无衬线字体。

### 禁止

- 正文使用装饰字体。
- 大面积使用手写体。
- 标题使用过于夸张的美术字体。

## 油画意象清单

以下意象贯穿卡牌、油画、死亡文化内容与个人经验：

花 · 信 · 镜子 · 戒指 · 钥匙 · 蜡烛 · 石榴 · 水面 · 门 · 书 · 鸟 · 船 · 丝线 · 灰烬 · 空椅子

每个意象在三层中展开：
1. **卡牌层**：在塔罗或雷诺曼中作为象征。
2. **物件层**：在「物的余生」中作为独立意象。
3. **文化层**：在「告别志」文章中作为具体的文化参照物。

## 动效原则

- **缓慢**：过渡时间以秒计，不以毫秒计。
- **克制**：不堆叠多种动效。
- **像画面在显影**：元素出现像暗部逐渐展现细节。
- **支持 `prefers-reduced-motion`**：检测用户系统偏好，减弱或停用动效。

### 禁止

- 惊吓效果。
- 夸张魔法粒子。
- 自动播放声音或音乐。
- 快速闪烁或高频变换。
- 无节制的视差滚动。

## 布局原则

- 页面留白充足，不拥挤。
- 内容区域宽度适中，不撑满屏幕。
- 用户阅读路径清晰：从上到下，从图像到文字。
- 油画可以出血（bleed）到边缘，但文字必须在安全区内。
- 移动端优先保证可读性，图像可以裁切但文字不能丢失。

## 暗色模式

- 项目本身就是暗色的——不提供浅色模式切换。
- 在明亮环境下仍可阅读，依赖足够的对比度（正文/背景 ≥ 4.5:1）。
